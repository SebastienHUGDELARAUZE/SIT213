## Application tag versionning

This project follow this specification for its version tag: `v.X.Y.Z`

| v.  | X.        | Y.             | Z      |
| --- | ---       | ---            | ---    |
| Tag | TP number | Version number | Status |

**Status** part can be:
 * `pre-release`
 * `release`
